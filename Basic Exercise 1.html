Goal of this exercise is to learn to use HTML.

What does HTML stand for?

HTML stands for HyperText Markup Language.

What is it used for?

We use it to build the different elements of our webpage.

What are those elements?

For example: paragraphs, headings, images tables, etc.

How do I create those elements?

The content that needs to be displayed, has to be put inside specific tags.
